API文档

以下是网页访问统计系统提供的所有API接口文档。

POST /api/statistics

上报访问数据，会自动获取访问者IP

请求参数 (JSON)

{
  "url": "https://example.com",      // 访问的完整URL
  "path": "/page1",                  // 请求路径
  "client_time": 1701234567890,      // 客户端时间戳（可选）
  "server_time": 1701234567000       // 服务器时间戳（可选）
}

请求示例

fetch('/api/statistics', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    url: window.location.href,
    path: window.location.pathname,
    client_time: Date.now()
  })
});

响应示例

{
  "code": 0,
  "msg": "success"
}
GET /api/overview

获取统计数据概览，需要登录

注意: 需要先登录获取 Cookie，Cookie 名称为 session_id

响应示例

{
  "code": 0,
  "msg": "success",
  "data": {
    "totalRecords": 1000,
    "uniqueIps": 150,
    "uniqueUrls": 50,
    "activeDays": 30,
    "todayCount": 25,
    "todayIps": 10
  }
}
GET /api/records

获取访问记录列表，需要登录

查询参数

?page=1&pageSize=25

响应示例

{
  "code": 0,
  "msg": "success",
  "data": {
    "records": [
      {
        "id": 1,
        "ip": "192.168.1.1",
        "url": "https://example.com",
        "path": "/page",
        "client_time": "2024-01-01 12:00:00",
        "server_time": "2024-01-01 12:00:01",
        "created_at": "2024-01-01 12:00:01"
      }
    ]
  },
  "pagination": {
    "page": 1,
    "pageSize": 25,
    "total": 100,
    "totalPages": 4
  }
}

错误码说明

0 成功
1 失败/错误
401 未登录或会话过期
429 请求过于频繁（限制100次/分钟/IP）
