API文档

以下是网页访问统计系统提供的所有API接口文档。

POST

提交统计数据

/api/statistics

提交网页访问统计数据，用于记录用户访问信息。

请求参数（JSON Body）

参数名类型必填说明
ipstring必填用户IP地址
urlstring必填访问的网页域名
pathstring必填访问的网页路径
client_timestring必填用户端时间
server_timestring必填服务器时间

响应示例

{ "code": 0, "msg": "success" }

调用示例

curl -X POST http://localhost:8000/api/statistics -H "Content-Type: application/json" -d '{"ip":"192.168.1.1","url":"example.com","path":"/home","client_time":"2024-01-01 10:00:00","server_time":"2024-01-01 10:00:05"}'
fetch('http://localhost:8000/api/statistics', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ ip: '...', url: '...', path: '...', client_time: '...', server_time: '...' })
})
.then(res => res.json())
.then(data => console.log(data));
import requests
response = requests.post('http://localhost:8000/api/statistics', json={...})
print(response.json())
GET

获取统计数据列表

/api/getStatistics?page=1&pageSize=25

获取统计数据列表，支持分页查询。需要登录状态。

{ "code": 0, "data": [...], "pagination": { "page": 1, "pageSize": 25, "total": 100, "totalPages": 4 } }
GET

获取数据概览

/api/overview

获取统计数据概览信息，包括总访问量、独立IP数等。需要登录状态。

{ "code": 0, "data": { "totalRecords": 100, "uniqueIps": 50, "uniqueUrls": 10, "activeDays": 7 } }

错误码说明

错误码说明
0成功
-1未登录或登录已失效
-2参数缺失或无效
-3记录不存在
-4操作失败
-5服务器内部错误
