统计管理

API文档

以下是网页访问统计系统提供的所有API接口文档。

POST 提交统计数据
/api/statistics
提交网页访问统计数据，用于记录用户访问信息。

请求参数（JSON Body）

参数名类型必填说明
ipstring必填用户IP地址
urlstring必填访问的网页域名
pathstring必填访问的网页路径
client_timestring必填用户端时间
server_timestring必填服务器时间

响应示例

{ "code": 0, "msg": "success" }

调用示例

curl -X POST http://localhost:8000/api/statistics \ -H "Content-Type: application/json" \ -d '{"ip":"192.168.1.1","url":"example.com","path":"/home","client_time":"2024-01-01 10:00:00","server_time":"2024-01-01 10:00:05"}'
fetch('http://localhost:8000/api/statistics', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ ip: '192.168.1.1', url: 'example.com', path: '/home', client_time: '2024-01-01 10:00:00', server_time: '2024-01-01 10:00:05' }) }) .then(res => res.json()) .then(data => console.log(data));
import requests data = { 'ip': '192.168.1.1', 'url': 'example.com', 'path': '/home', 'client_time': '2024-01-01 10:00:00', 'server_time': '2024-01-01 10:00:05' } response = requests.post('http://localhost:8000/api/statistics', json=data) print(response.json())
$url = 'http://localhost:8000/api/statistics'; $data = [ 'ip' => '192.168.1.1', 'url' => 'example.com', 'path' => '/home', 'client_time' => '2024-01-01 10:00:00', 'server_time' => '2024-01-01 10:00:05' ]; $ch = curl_init(); curl_setopt($ch, CURLOPT_URL, $url); curl_setopt($ch, CURLOPT_POST, true); curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data)); curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/json']); curl_setopt($ch, CURLOPT_RETURNTRANSFER, true); $response = curl_exec($ch); curl_close($ch); echo $response;
GET 获取统计数据列表
/api/getStatistics?page=1&pageSize=25
获取统计数据列表，支持分页查询。需要登录状态。

请求参数（Query）

参数名类型必填说明
pageinteger可选页码，默认1
pageSizeinteger可选每页数量，默认25，范围1-100

响应示例

{ "code": 0, "data": [ {"id": 1, "ip": "192.168.1.1", "url": "example.com", "path": "/home", ...} ], "pagination": { "page": 1, "pageSize": 25, "total": 100, "totalPages": 4 } }
GET 获取数据概览
/api/overview
获取统计数据概览信息，包括总访问量、独立IP数等。需要登录状态。

响应示例

{ "code": 0, "data": { "totalRecords": 100, "uniqueIps": 50, "uniqueUrls": 10, "activeDays": 7, "todayRecords": 15, "todayUniqueIps": 8 } }
GET 获取图表统计数据
/api/getStats
获取图表分析所需的数据，包括IP分布、时间分布、访问趋势。需要登录状态。

响应示例

{ "code": 0, "data": { "ipDistribution": [{"ip": "192.168.1.1", "count": 50}, ...], "hourDistribution": [{"hour": 0, "count": 5}, ...], "dailyTrend": [{"date": "2024-01-01", "count": 20}, ...] } }
DELETE 删除单条记录
/api/statistics/:id
删除指定的统计记录。需要登录状态。

请求参数

参数名类型位置说明
idintegerURL路径要删除的记录ID
forcestringQuery（可选）设为"1"时为强制删除（不记录确认日志类型）

响应示例

{ "code": 0, "msg": "删除成功" }

调用示例

# 普通删除 curl -X DELETE http://localhost:8000/api/statistics/1 # 强制删除（Shift+点击效果） curl -X DELETE "http://localhost:8000/api/statistics/1?force=1"
// 普通删除 fetch('http://localhost:8000/api/statistics/1', { method: 'DELETE' }) // 强制删除 fetch('http://localhost:8000/api/statistics/1?force=1', { method: 'DELETE' })
POST 批量删除记录
/api/statistics/batch-delete
批量删除多条统计记录。需要登录状态。

请求参数（JSON Body）

参数名类型必填说明
idsarray[integer]必填要删除的记录ID数组
forceboolean可选设为true时为强制删除

响应示例

{ "code": 0, "msg": "成功删除 5 条记录" }

调用示例

fetch('http://localhost:8000/api/statistics/batch-delete', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ ids: [1, 2, 3, 4, 5], force: false }) })
GET 获取操作日志
/api/logs?page=1&pageSize=25
获取操作日志列表，支持分页查询。需要登录状态。

请求参数（Query）

参数名类型必填说明
pageinteger可选页码，默认1
pageSizeinteger可选每页数量，默认25，范围1-100

响应示例

{ "code": 0, "data": [ {"id": 1, "operation_type": "login", "target_id": null, "session_id": "abc...", "ip": "::1", "created_at": "2024-01-01T10:00:00Z"} ], "pagination": {"page": 1, "pageSize": 25, "total": 50, "totalPages": 2} }
错误码说明
错误码说明
0成功
-1未登录或登录已失效
-2参数缺失或无效
-3记录不存在
-4操作失败
-5服务器内部错误
