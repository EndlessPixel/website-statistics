API文档

POST /api/statistics

描述

上报访问统计数据

请求参数

{
  "ip": "192.168.1.1",        // 用户IP地址
  "url": "https://example.com", // 访问的URL
  "path": "/page",             // 路径
  "client_time": 1699999999,   // 用户端时间戳(秒)
  "server_time": 1699999999    // 服务器时间戳(秒)
}

成功响应

{
  "code": 0,
  "msg": "success"
}

失败响应

{
  "code": -1,
  "msg": "参数错误"
}
GET /api/records?page=1&pageSize=25

描述

获取访问记录列表

请求参数

  • page - 页码，默认1
  • pageSize - 每页数量，默认25，范围1-100
GET /api/stats

描述

获取统计数据（用于图表）

GET /api/logs?page=1&pageSize=25

描述

获取操作日志

POST /api/statistics/batch-delete

描述

批量删除访问记录

请求参数

{
  "ids": [1, 2, 3],  // 要删除的记录ID数组
  "force": false      // 是否强制删除（跳过确认）
}
GET /api/query-ip?ip=x.x.x.x

描述

查询IP归属地信息（带缓存）

GET /api/repo

描述

获取GitHub仓库信息

GET /api/version

描述

获取Git版本信息

调用示例

cURL

curl -X POST http://localhost:8000/api/statistics \
  -H "Content-Type: application/json" \
  -d '{
    "ip": "192.168.1.1",
    "url": "https://example.com",
    "path": "/page",
    "client_time": '"$(date +%s)"',
    "server_time": '"$(date +%s)"'
  }'

JavaScript (Fetch)

fetch('http://localhost:8000/api/statistics', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    ip: '192.168.1.1',
    url: window.location.href,
    path: window.location.pathname,
    client_time: Math.floor(Date.now() / 1000),
    server_time: Math.floor(Date.now() / 1000)
  })
});

Python (requests)

import requests
import time

data = {
    'ip': '192.168.1.1',
    'url': 'https://example.com',
    'path': '/page',
    'client_time': int(time.time()),
    'server_time': int(time.time())
}

response = requests.post(
    'http://localhost:8000/api/statistics',
    json=data
)
print(response.json())

PHP

<?php
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 'http://localhost:8000/api/statistics');
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    'ip' => '192.168.1.1',
    'url' => 'https://example.com',
    'path' => '/page',
    'client_time' => time(),
    'server_time' => time()
]));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json'
]);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$response = curl_exec($ch);
curl_close($ch);
print_r(json_decode($response, true));
?>
