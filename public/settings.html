GitHub API

仓库信息 · 贡献图 · 项目元数据获取

在 GitHub 设置中创建 Personal Access Token (classic)，无需额外权限即可提升配额

认证方式 请求限制 适用场景
未认证 (IP限制) 60 次/小时 仅测试或极低流量站点
Personal Access Token 5,000 次/小时 个人开发者/普通项目(推荐)
GitHub App 安装令牌 5,000 ~ 15,000 次/小时* 企业级集成/高并发场景
OAuth App Token 同授权用户 Token 限制 用户授权登录类应用

📈 进阶配额说明：

  • 使用安装访问令牌 (Installation Access Token) 认证的 GitHub App，默认限制为 5,000次/小时。
  • 若安装在 GitHub Enterprise Cloud 组织上，限制提升至 15,000次/小时。
  • 其他 Enterprise Cloud 之外的安装，速率限制根据用户/仓库数量扩展，最高可达 12,500次/小时。
  • 💡 对于绝大多数统计需求，Personal Access Token (5k次/小时) 已完全足够，可缓存数据进一步降低调用频率。
令牌仅用于提高请求速率，不会记录您的私有仓库数据，请妥善保管 Token。
